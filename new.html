Lire le README.md /!\Au lancé GoldenSun MAp theme song se lance.

Pour compiler le programme faire la commande make (dasn source)
ensuite pour executer ./projet map.osm :

Vous pouvez vous deplacer et zoomer sur la map :
Les touches /!\ : "a "= zoom avant " z"=zoom arriere "e"=inclinaison "r"=declinéson "o"=centrage de map (si perdu dans le a map) "p"=debut d'affichage des balises names "m"= ne plus afficher les balises names.

Pour fermer le programme appuyer sur ECHAP -> une image au format BMP se crée automatiquement :sortie.bmp

./projet map.osm --svg permet de créer une image au format SVG.faire make svg avant chaque usage /!\

dans style/style.xml on a la possibilité de changer les couleurs en changant les couleurs RGBA

Si une map est trop grande preferer l'option : ./projet map.osm --zoom car elle zoom et se deplace sur le BMP crée.

l'api overpass permet de generer puis afficher une map a l'aide des valeurs de bounds /.projet 2.36589 48.85280 2.36924 48.85410

./projet map.osm --past geenre une map façon carte au trésor (les caclues sont fait de sorte a ce que la terre soit plate)

./projet map.osm --futur simule la map version futur.

Une Api permettra de donner la meteo(seulement dans le terminal). Nous utilisons une Api gratuite ,l'api ne fonctionne malheureusement que sur Paris.
